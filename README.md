

🤖☕ Robot Barista
A simple interactive Python program that simulates a coffee shop ordering system. The virtual barista greets customers, takes their order, and calculates the total price based on the number of cups.
Features

Personalized greeting using the customer's name
Displays a menu with 3 drink options
Validates the order (only accepts items 1–3)
Calculates total price based on cups ordered
Friendly and conversational interaction
